Project Title:
Ganesh Event Management - Premium Event Planning Platform

Short Description:
A high-end, responsive web application for a luxury event management boutique based in Bhind, Madhya Pradesh. This platform showcases a premium portfolio, cinematic video highlights, and an interactive contact system designed to convert visitors into clients for weddings, corporate galas, and private celebrations.

Key Features:
Premium Visual Experience: Utilizes fluid typography and glassmorphism design trends to reflect a luxury brand identity.

Dynamic Portfolio Gallery: A filtered masonry grid that allows users to explore past events (Weddings, Corporate, Birthdays) with detailed case studies.

Cinematic Video Showcase: Integrated video gallery featuring high-definition event highlights with optimized performance across all devices.

Fully Responsive & Adaptive: Custom-built media queries and flexible layouts ensure a seamless experience on mobile, tablets, laptops, and 4K desktops.

Interactive Engagement: Features a modern FAQ accordion system, an integrated Google Maps location for the Bhind studio, and a prominent WhatsApp direct-action button.

High Performance: Optimized with framer-motion for smooth entry animations and Lucide React for lightweight, scalable iconography.

Tech Stack:
Frontend: React.js, TypeScript

Styling: Tailwind CSS (Mobile-first responsive design)

Animations: Framer Motion

Icons: Lucide React

Deployment: Vercel
